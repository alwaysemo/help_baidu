# HELP_BAIDU 让我帮你百度一下

让我帮你百度一下是一个专门为伸手党准备的网站。它用动画的方式教他人使用搜索引擎，你只需要输入一个问题，将获得的链接发给好友即可。
该项目后续将会完善 Google、Bing 等搜索引擎页面。

### 预览

在线预览地址：https://baidu.alwaysemo.com
![预览图](https://github.com/alwaysmeo/help_baidu/blob/master/src/assets/images/preview.png)

### 依赖安装

```sh
bun install
```

### 运行

```sh
bun dev
```

### 编译

```sh
bun run build
```
